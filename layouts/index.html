the page layout

Editorial spread layouts for the Marina Fragoso magazine — page anatomy modelled on the GQ feature and the Santal layout system. Every page carries the same chrome: the kicker (her signature, the clickable site logo) at the top, and a footer with the site menu and the page folio. The page-flip motion comes later from the flipbook engine; these are the static designs it will turn.
